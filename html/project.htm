Line Segment Intersection Using Sweepline

HOW TO USE
Run: Finds all intersection points
Step: Incrementally steps through algorithm
Reset: Clears all line segments
Debug: Toggles debug information
Red line: Sweep line
Blue lines: Default color
Black lines: Lines intersecting the sweep line (in the status structure)
Orange lines: Direct neighbors to the line segment(s) for the current event point
Small black dots: Points of intersection with the sweep line
Red dots: Found intersection points
Large black dot: First point of a new line segment
